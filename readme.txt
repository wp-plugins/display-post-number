== Description ==

This 'Display Post Number' is a plugin that you can set the number of articles to be displayed in the dashboard. 
It is a plugin to support only Japanese.

== Installation ==

1. Upload `display-post-number` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
